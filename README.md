# -Syntecxhub_-ipl_data_analysis
It analyzes IPL match and player datasets to discover key insights, player statistics, and team performance trends.
The goal of this project is to perform data analysis on IPL matches to understand:

Top run scorers and consistent performers

Player strike rates and efficiency

Team win rates and performance comparison

Per-season performance trends

The analysis helps identify how player and team performance evolved across multiple IPL seasons.
